# Sample simulated data for exposure-response.

Sample simulated data for exposure-response.

## Usage

``` r
exposure.response.sample
```

## Format

A data frame with columns:

- dose:

  Dose levels used for simulation of pharmacokinetics

- exposure:

  Simulated exposure

- response:

  Simulated pharmacodynamic response
